About the Studio

We make things that move.

Resn is an independent interactive studio based in Wellington, New Zealand. Since 2004 we have been building the kind of work that lives on a single URL and a long screen — sites, identities, motion systems, and live experiments for the brands and inventors we admire.

We are deliberately small. We write our own tools, we draw our own frames, and we ship the whole thing. The studio is part workshop, part theatre, part observatory.
