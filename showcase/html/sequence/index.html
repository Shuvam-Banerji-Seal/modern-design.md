For developers

A wallet in under thirty lines.

Type-safe SDKs, deterministic addresses, and a hosted signer that you can theme, scope, and revoke.

// 1. Initialise the wallet client
const wallet = await sequence.initWallet({
  network: "polygon",
  projectAccessKey: process.env.SEQUENCE_KEY,
});

// 2. Sign in with a passkey
const session = await wallet.signIn({ idToken });

// 3. Send a sponsored transaction
const tx = await session.sendTransaction({
  calls: [{ to, value, data }],
  sponsored: true,
});
