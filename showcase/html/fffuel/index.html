🔥 a small thing we made — check it out

fffuel is a small collection of color utilities and free SVG makers for gradients, patterns, textures, shapes & backgrounds.

🚀 A pocket library of in-browser generators for designers and front-end folks — open one, tweak a few sliders, copy the SVG.

🤹‍♂️ Every tool exposes a few knobs so the output stays yours, not a generic preset.

✨ SVG is small, scales to any size, and travels well across browsers, slides and screenshots.

🔧 Need a PNG instead? One-click rasterizer turns the result into a flat image at any DPI.

🖼️ Drop a generated SVG into a background-image and you've got a tileable cover in a line of CSS.

🎨 A few color helpers sit alongside — pick a swatch, build a ramp, encode the value.
